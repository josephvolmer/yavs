# YAVS Documentation

Comprehensive documentation for YAVS (Yet Another Vulnerability Scanner).

## Main Documentation

- **QUICKSTART.md** - Quick start guide for getting up and running
- **AI_PROVIDER_GUIDE.md** - Guide for using AI features with different providers
- **NATIVE_CONFIGS.md** - Using native scanner configuration files
- **OUTPUT_SCHEMAS.md** - Documentation of output format schemas
- **WORKFLOWS_OVERVIEW.md** - Overview of common workflows
- **SUMMARIZE_BEHAVIOR.md** - AI summarization feature documentation
- **PRODUCTION_CLI.md** - Production usage and CLI reference
- **MAKEFILE_GUIDE.md** - Using the Makefile for development
- **KNOWN_ISSUES.md** - Known issues and limitations

## Directory Structure

```
docs/
├── development/     # Development and implementation documentation
├── examples/        # Example JSON outputs
├── schemas/         # JSON schemas for output formats
└── images/          # Images and screenshots
```

## Getting Help

For interactive documentation within the CLI, use:
```bash
yavs man
```

For the main README and installation instructions, see the root README.md.
